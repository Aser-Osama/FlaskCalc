# Flask_Calc
A simple calculator made using Python and the Flask framework.

## Table of Contents?
- [Installation?](#Installation)
- [Usage](#Usage)
- [Demonstration](#Demonstration)

## Prerequisites
- Flask, 3.12.2+
- Python, 3.0.2+
- Werkzeug, 3.0.1+

## Installation
1. Clone the repository:

```
git clone https://github.com/Aser-Osama/FlaskCalc.git
```
2. Go to directory
```
cd FlaskCalc
```
3. Run the program:
```
flask --app app run 
```
